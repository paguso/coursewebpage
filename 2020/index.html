Sobre o curso

Ementa

Algoritmos e estruturas de dados destinados ao processamento de grandes volumes de dados (big data), em particular dados coletados em fluxo contínuo (stream), para os quais as técnicas tradicionais baseadas na armazenagem dos dados em memória principal não são adequados. Os métodos aqui apresentados consistem na construção de sumários e índices dos dados, permitindo consultas aproximadas com garantias teóricas de erro, em memória sublinear. Estas técnicas visam a fornecer suporte para análise de dados e tomada de decisões em tempo real.
